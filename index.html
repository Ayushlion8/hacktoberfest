Rock Paper Scissors

This is a command-line interface to play the best game of all time: Rock, Paper, Scissors.

Rules of the Game

  • Rock wins against scissors.
  • Scissors win against paper.
  • Paper wins against rock.

    • How to play

  • Clone the project. It's available on GitHub.
  • Run with: python rockpaperscissor.py
  • Follow the prompts.
    0 = Rock
    1 = Paper
    2 = Scissors
